Main HTML structure

Doctype is a head that the browser uses to identify the type of file presented in the page.
After declaring DOCTYPE, we have the html opening and closing brackets representing all that we write after as html elements.
The head is the part of the html that contains the meta tags, links, scripts, and stylesheets.
The body is the part of the html that contains the content of the page.
Most website have a header, a main, and a footer, html has tags for those.

Information about HTML tag structuring

Attributes inside the tag give the tag some custom information.
The img attribute allows us to display the images
The src is what it uses to find the image.
While the alt is alternative text which is used when the image cannot be displayed
Most html tags are container tags, everything else is just a tag.
The html tag is the parent of the head and body tags.
The head and body tags are the child of the html tag.
The head and body are sibling tags.
The img and paragph tags are grandchilds of the html tag, and the child tags of body tags.
Some tags are container tags while others are content tags.

Information about Meta Tags

A meta tag is basically a tag that defines information about the our file.
Character set is the encoding of the file. Charset is used to specify the character set of the file.
UTF-8 is the default encoding for html 5.

Name tells the meta tag thae type of information given to it. And content is the information.

Main HTML Tags

Headers are like the same as github ###### since they just give a Heading to the file

Styling in HTML using CSS

You mainly use CSS to style HTML content.
Basic sytling includes adding a text color or background color.
If you give the paragraph tag the style attribute and add the attribute value of color and the color you want it to be, it will change the color of the paragraph text.
You can also change the background color of the paragraph.
The styling can be added to any tags that include some text content.

Links in HTML

Links are available to be used in html by using the html tag "a" with a href attribute that holds the URL.
You can add a Link text by adding the text outside the the attrbute value but before the closing tag.
You dont have to only have simple link text you can next elements inside the "a" tag.
Use the target attribute inside the tag to specify where the link opens to.
To link between pages of the website you can use href attribute with a relative URL.
