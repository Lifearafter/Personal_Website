Main HTML structure

Doctype is a head that the browser uses to identify the type of file presented in the page.
After declaring DOCTYPE, we have the html opening and closing brackets representing all that we write after as html elements.
The head is the part of the html that contains the meta tags, links, scripts, and stylesheets.
The body is the part of the html that contains the content of the page.
Most website have a header, a main, and a footer, html has tags for those.

Information about HTML tag structuring

Attributes inside the tag give the tag some custom information.
The img attribute allows us to display the images
The src is what it uses to find the image.
While the alt is alternative text which is used when the image cannot be displayed
Most html tags are container tags, everything else is just a tag.
The html tag is the parent of the head and body tags.
The head and body tags are the child of the html tag.
The head and body are sibling tags.
The img and paragph tags are grandchilds of the html tag, and the child tags of body tags.
Some tags are container tags while others are content tags.

Information about Meta Tags

A meta tag is basically a tag that defines information about the our file.
Character set is the encoding of the file. Charset is used to specify the character set of the file.
UTF-8 is the default encoding for html 5.

Name tells the meta tag thae type of information given to it. And content is the information.

Main HTML Tags

Headers are like the same as github ###### since they just give a Heading to the file
The paragh tags hold most of the content of the page such that it allows for articles to be written.

Styling in HTML using CSS

You mainly use CSS to style HTML content.
Basic sytling includes adding a text color or background color.
If you give the paragraph tag the style attribute and add the attribute value of color and the color you want it to be, it will change the color of the paragraph text.
You can also change the background color of the paragraph.
The styling can be added to any tags that include some text content.

Links in HTML

Links are available to be used in html by using the html tag "a" with a href attribute that holds the URL.
You can add a Link text by adding the text outside the the attrbute value but before the closing tag.
You dont have to only have simple link text you can next elements inside the "a" tag.
Use the target attribute inside the tag to specify where the link opens to.
To link between pages of the website you can use href attribute with a relative URL.

Images and Videos

Images in html can be displayed simply using img tag with the src attribute.
It is recommended to have a alt attribute for the image so that in case of image not found there is some thing to be displayed.
Video is also very similar in the fact that you can display a video using a video tag.
Videos also have a source tag that is used to display the video if it is in the files.
You can also display a video from a website by using the iframe.
Vidoes also have other attributes like controls, autoplay, and loop.
You can also resize any image or video by defining the witdh and/or height attributes.
If only one attribute is defined the other will be set by html depending on the aspect ratio

Lists

Lists are used to display a list of items.
Lists can be ordered or unordered depending on the tag used.
Unordered lists are used with the ul tag.
Ordered lists are used with the ol tag.
Lists items displayed with the li tag.
You can nest tags within li tags.
With il tag you can also add a type attribute to the list which defines the type of character
A, a, i, I, and the default 1, are the values that can be assigned to the Type attribute.
You can nest lists.

Containers

Containers in HTML are extremely useful for styling and organization.
Two of the most used containers being divs and spans.
Divs are block containers that instead of taking just a line of text, they take up a block of space.
Spans are inline containers that take up a line of text.
There are other containers, but mostly divs and spans are used.
You can use the class tag on any container to give it a name, that can then be used by the stylesheets.

Input Forms and Buttons

Input forms are used to collect information from the user.
Any input for can be used with the input tag and the appropriate type.
The type attribute is used to define the type of input.
There are many different types, you can find more information here.

Meta Tags

Tags that are used to basically define the meta data of the data for the webstie.
Tags like charset, name, content, and http-equiv are used to define the meta data.
You can also use the attributes viewport to help display the website properly on different devices.
Theres also the meta tag that is used to define the description of the website.
And then theres the meta tag that is used to define the keywords of the website.
And then theres the meta tag that is used to define the author of the website.
